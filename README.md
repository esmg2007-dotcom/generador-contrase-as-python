# Generador de Contraseñas Seguras

Proyecto de Programación 1 desarrollado en Python.

## Descripción

Este programa permite generar contraseñas seguras de acuerdo con las opciones seleccionadas por el usuario. El usuario puede ingresar la longitud de la contraseña y decidir si desea incluir letras, números y símbolos.

## Herramientas utilizadas

- Python
- Visual Studio Code
- GitHub
- PowerPoint para el diagrama de flujo

## Funcionalidades

- Solicitar la longitud de la contraseña.
- Permitir incluir letras, números y símbolos.
- Validar que el usuario seleccione al menos una opción.
- Generar una contraseña aleatoria.
- Permitir generar otra contraseña mediante un ciclo.

## Estructuras utilizadas

- Condicionales if
- Bucle for
- Bucle while
- Funciones
- Comentarios en el código
